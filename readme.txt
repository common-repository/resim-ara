Resim bulmanızı hızlandıracak, kolaylaştıracak resim bulma eklentisi.

== Description ==

Başlığınızı alıp, resim bulma arama motoru kısmında arayan böylece size zaman ve emek kazancı sağlayan resim bulma eklentisi

== Installation ==

Kurulum oldukça basittir

1. Eklenti dosyalarını `/wp-content/plugins/alikaan` klasörüne kopyala ya da direkt olarak wordpress eklenti bölümünden kurulumunu yapın.
2. Wordpress yönetim paneli içerisinden Eklentiler bölümünden eklentiyi etkinleştirin.
3. Yazı düzenleme içerisinde sağ üstteki panelden "ara" butonuna basın, isterseniz kelimeyi değiştirebilirsiniz.
4. Klasör izini 755 olabilir dosyalar 644 olabilir kendinize göre denersiniz.
